---
title: Install using a Binary Release
---

### Darwin (macOS)

```bash
# <RELEASE_VERSION> is the release version i.e. 0.5.14

# First, fetch the latest release
wget https://github.com/CandlePlatforms/go-candle/releases/download/<RELEASE_VERSION>/candle-darwin-amd64.tar.gz

# Next, extract it
tar -zxvf candle-darwin-amd64.tar.gz

# Finally, move it to the appropriate directory
mv candle-darwin-amd64/* /usr/local/bin/
```
